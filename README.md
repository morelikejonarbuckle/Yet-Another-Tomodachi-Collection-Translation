# Yet-Another-Tomodachi-Collection-Translation
A improved translation of Tomodachi Collection.
The foundation of this translation relies on Deepl, **an AI translator.** It's the most accurate one I can find. Otherwise, I used **ChatGPT** for phrases that don't translate well. Complain about it here: https://gbatemp.net/threads/complain-about-morelikejonarbuckle0s-tomodachi-collection-translation-using-ai-here.681360/
### If you hate AI, use tools in the wiki as a template to make another translation
Some phrases were left out because they were too long in English, or I thought of some of my own. Hopefully they don't break immersion too much...

# About
I got bored one day and impulsively tried to finish jjjewel's translation.
Some text might get cut off when you play this ROM hack. Might set up a google form to submit errors in case some people don't want to sign up for a Github account...

Some people have starred this for some reason, if they want to try translating their own copy here's a neat guide to some textures: https://gbatemp.net/threads/tomodachi-collection-textures-to-replace.675991/
Use this version of Tinke or else the textures refuse to load.
https://github.com/R-YaTian/TinkeDSi/actions/runs/22517582323

This MSBT editor works to edit BMG (the text file of the game):
https://gitlab.com/AeonSake/msbt-editor/-/releases/
For News & Job and maybe Song BMGs you might have to use NitroPaint. Previous versions of that editor were breaking the translation.
